# Employee Management System

A web-based application built using **Python and Django** to manage employee records, including adding, updating, viewing, and deleting employee information.

This project demonstrates backend development, database integration, and CRUD operations.

## Features
- Add new employee records
- Update employee information
- Delete employee records
- Store and manage employee data using a database
- Perform CRUD operations through Django backend

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite Database
  
## Project Structure

EmployeeManagement/
│
├── EmployeeManagement/ # Django project settings
├── Office-Employee-details/ # Django app
├── templates/
├── db.sqlite3
└── manage.py

## How to Run the Project

1. Clone the repository

git clone https://github.com/Kritikagupt/employee-management-system.git


2. Navigate to the project directory

cd employee-management-system


3. Install dependencies

pip install django


4. Run the development server

python manage.py runserver


5. Open in browser

http://127.0.0.1:8000/
